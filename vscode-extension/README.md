# apitest-gen (VS Code extension)

Runs the [apitest-gen](https://github.com/chaitanyasantapur/scriptgenerationtool) CLI from the Command Palette: validate OpenAPI, generate REST tests, or add a GraphQL smoke stub.

Open a workspace folder, then use commands starting with **apitest-gen:**.

See the main repository README for CLI flags and environment variables (`API_BASE_URL`, etc.).

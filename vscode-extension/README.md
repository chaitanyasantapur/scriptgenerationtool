# apitest-gen (VS Code extension)

Runs the [apitest-gen](https://github.com/chaitanyasantapur/scriptgenerationtool) CLI from the Command Palette: validate OpenAPI, generate REST tests, GraphQL smoke stub, **WebdriverIO Page Object** scaffold, and **WebdriverIO spec** scaffold.

Open a workspace folder, then use commands starting with **apitest-gen:**.

See the main repository README for CLI flags and environment variables (`API_BASE_URL`, etc.).
